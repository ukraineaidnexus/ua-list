title: "Contribute directly"
---

If you're comfortable with git and GitHub, you can skip the submission forms and edit entries yourself.

## How it works

1. Fork [ua-list-data](https://github.com/ukraineaidnexus/ua-list-data), it holds the organisation, volunteer, fundraiser, creator, and follow entries as YAML files.
2. Add or edit a file under `data/orgs/`, `data/fundraisers/`, `data/creators/`, or `data/follow/`. The repo's [README](https://github.com/ukraineaidnexus/ua-list-data#readme) has the schema for each entry type.
3. Open a pull request against `main`.
4. I review it and merge if it checks out. Nothing goes live automatically, every change is read by a human before it reaches the site.

## What this covers

Orgs, volunteers, fundraisers, creators, and accounts to follow. This data is released under CC0 (public domain), so you're also free to fork it, reuse it, or build your own tools on top of it. No permission needed.

## What it doesn't cover

The scammer register isn't open to public pull requests. If you've got evidence of a scam, use the [report a scammer](/submit/?type=scammer) form instead. Evidence gets checked before anything is published.

## Not comfortable with git?

Use the [submission forms](/submit/) instead. Same review process, just a form instead of a pull request.
